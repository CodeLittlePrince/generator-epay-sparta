> Epay前台前端脚手架

## 介绍
Epay前台前端脚手架（PC、H5、微信小程序）

## 使用

```bash
npm install -g yo generator-epay-sparta
```

新建项目:

```bash
yo epay-sparta
```

在已经存在的项目上新增单测功能:
```bash
yo epay-sparta add test:unit
```
